# DomeLight Device Map

This folder contains the stored IR command map for a **HELPLEX LED Puck Lights with Remote (3-pack RGBW battery puck lights)**.

Source product page: https://www.amazon.com/dp/B09XPWDGDJ

## Stored IR Codes

Protocol is `NEC` and each command is `32` bits.

| Index | Command Name   | Hex Value |
| --- | --- | --- |
| 0 | Off | `00FF629D` |
| 1 | Purple | `00FF7A85` |
| 2 | Green | `00FF9867` |
| 3 | Rainbow | `00FF22DD` |
| 4 | Blue | `00FFB04F` |
| 5 | On | `00FFA25D` |
| 6 | Flash | `00FF02FD` |
| 7 | Teal | `00FF18E7` |
| 8 | Red | `00FF6897` |
| 9 | Orange | `00FF30CF` |
| 10 | Sun | `00FF10EF` |
| 11 | Yellow | `00FF42BD` |
| 12 | Aqua | `00FF4AB5` |
| 13 | Light Aqua | `00FF38C7` |
| 14 | Sea Foam | `FF18E7` |
| 15 | 30M | `00FFE21D` |
| 16 | 60M | `00FFC23D` |
| 17 | Brighter | `00FFE01F` |
| 18 | Less Bright | `00FFA857` |
| 19 | White | `00FF906F` |
| 20 | Violet | `00FF5AA5` |
| 21 | Bright Purple | `00FF52AD` |
